Contributing Decals

To contribute your own decals to the catalog, you will need to send them to Colson. You can find him by joining his Discord server!

Decals will need to meet the following criteria:

- File size cannot exceed 256kb
- Names will need to be in PascalCase and not exceed 25 characters in length
- Decals need to be sent in a .zip file to simplify the integration process.

If possible, try to include a HTML text snippet like this for your decals, as it will help immensely with integrating decals into the system.
