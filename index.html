Update 12

July 29th, 2023

The website has had quite alot of changes behind the scenes. I'm gradually beginning the process of stripping Python out with the eventual goal of more Java everywhere, because who doesn't like Java?

Some UI changes were made to remove bootstrapper code. It might not look as pretty, but most pages will load faster now. The exception is for the RNG pages, which I won't touch until after the contest ends.

A decals catalog is now available! This will expand with new decals over time - feel free to send some my way through Discord if you want to see them included!

It was becoming noticeable that people had a hard time reading the navigation menu, which contained everything people were looking for, but they simply didn't see that it was there. As a result, it is now much bigger.

The description for the random tank generator has been changed. Additionally, the outdated changelog page is no longer visible through the credits section.



Update 10

July 8th, 2023

A new rulesheet has been added! Refer to this page for additional info about the contest.

Armored Trains now allow for wheels to be file edited.



Update 9

July 8th, 2023

Finally outpaced Satisfactory in update numbers!

There has been an update to the rules for Toyotas and IFVs

Toyotas can now fall under the open-top classification, meaning you're able to ignore some space limitations. IFVs are now also allowed to be wheeled, meaning their tracks can be file edited to be invisible.

Remember that cheating is still not permitted. Things such as turret boosters, hiding compartments inside the vehicle, and non-permitted file editing isn't allowed.
